installation
==============
- apt-get install metasploit-framework
- /opt/metasploit-framework/bin/msfconsole
- /opt/metasploit-framework/bin/msfdb
- msfdb init

start the program
====================
- service postgresql start
- service metasploit start
- msfconsole

update
========
- msfupdate
- apt-get dist-upgrade
